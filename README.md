CollabCode- a full-featured collaborative development platform with real-time coding!

Key Features-
- Multi-user code editing with instant synchronization  
- Supports multiple programming languages (Python, Java, etc.)  
- Seamless room-based collaboration via shareable links  
- Integrated video and audio calling for live discussions  
- Real-time chat for quick messaging alongside coding  
- Collaborative whiteboard for brainstorming and explanations  
- Multiple themes for better user experience
